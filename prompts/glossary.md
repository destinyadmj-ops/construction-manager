# Glossary (Master Hub)

プロジェクト内で使う用語・略語を固定します。プロンプトの誤解を減らすため、短くても良いのでここに集約します。

## 用語
- **予定 / Event**: `CalendarEvent` に相当
- **作業 / Work**: `WorkEntry` に相当
- **連携 / Integration**: 外部サービスアカウント（`IntegrationAccount`）
- **リマインド / Reminder**: BullMQ の `reminders` キューで扱うジョブ

## 省略語
- **DB**: Postgres
- **Redis**: BullMQ のバックエンド
- **PWA**: Progressive Web App
